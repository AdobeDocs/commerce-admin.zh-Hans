# Adobe Commerce生态系统

Adobe提供丰富的业务和技术资源、自助工具和服务，帮助您获得成功。

| 生态系统资源 | 描述 |
| ------------------ | ----------- |
| [产品文档](https://experienceleague.adobe.com/docs/commerce.html?lang=zh-Hans) | 商家、设计人员和开发人员可以使用该文档深入了解insight如何运用[!DNL Commerce]的强大功能发展业务。 |
| [Commerce资源库](https://business.adobe.com/cn/resources/main.html?Products+%26+Services=Commerce%252CCommerce%2520Cloud) | 访问Ebooks、Commerce白皮书和网络研讨会，获得insight和在线业务资源方面的专家意见，帮助开发和改进您的商店。 |
| [Adobe产品安全事件响应](https://helpx.adobe.com/cn/security.html) | 加入安全警报注册表以获取有关潜在漏洞和[安全最佳实践](https://www.adobe.com/content/dam/cc/en/security/pdfs/Adobe-Magento-Commerce-Best-Practices-Guide.pdf)的最新信息。 |
| [社区论坛](https://community.magento.com/) | 通过论坛，可访问由分享技巧、技巧和支持的专业Adobe Commerce和Magento Open Source爱好者组成的网络。 |
| [博客](https://business.adobe.com/blog/) | 请查看博客以了解有关新增功能、趋势、最佳实践、即将举行的活动等的最新信息。 |
| [活动和网络研讨会](https://www.adobe.com/events.html) | 密切关注即将举行的活动和网络研讨会的日历。 [!DNL Commerce]生态系统中始终发生一些情况。 |
| [[!DNL Commerce Marketplace]](https://marketplace.magento.com/) | 访问[!DNL Commerce]新一代数字商务解决方案市场。 [!DNL Commerce Marketplace]为商户提供精选的解决方案，同时为合格的开发人员提供工具、平台和最理想的位置，促进业务的蓬勃发展。 |
| [培训和认证](https://learning.adobe.com/catalog.html?solution=Adobe%20Commerce) | 了解如何释放Adobe Commerce和Magento Open Source的强大功能和灵活性。 为您企业中的每个角色（包括营销人员、设计人员、开发人员等）提供培训。 经验丰富的[!DNL Commerce]专业人员可以通过获得[!DNL Commerce]认证来验证其真实技能。 |
| [合作伙伴](https://business.adobe.com/cn/products/magento/partners.html) | Adobe合作伙伴致力于帮助您获得成功，并为内部部署和云解决方案提供自定义集成、一流的客户体验、战略营销举措，以及性能和可扩展性方面的专家优化。 |
| [Adobe Commerce Managed Services](https://business.adobe.com/cn/products/magento/fully-managed-service.html) | Adobe Commerce Managed Services团队帮助商家和解决方案合作伙伴获得最大成功。 这些专家提供从架构规划到部署后的全面分析和最佳实践建议。 |
