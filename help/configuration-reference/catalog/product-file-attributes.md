# [!UICONTROL Catalog] > [!UICONTROL Product File Attributes]

## [!UICONTROL Configure Allowed File Types and Size]

![产品文件属性](./assets/file-validation.png)

| 字段 | [作用域](../../getting-started/websites-stores-views.md#scope-settings) | 描述 |
|--- |--- |--- |
| [!UICONTROL Allowed File Extensions] | 全局 | 允许的文件类型的逗号分隔列表，如`pdf,doc,docx,txt`。 留空将使用默认允许的文件类型：pdf、doc、docx、xls、xlsx、ppt、pptx、txt、csv、zip。 |
| [!UICONTROL Maximum File Size] | 全局 | 允许的最大文件大小（以MB为单位），如`20.0`。 默认限制为16 MB。 允许的最大文件大小为31 MB。 |
